# URGENT: Environment Variables Setup
# আপনার system কাজ না করার কারণ - Environment Variables missing!

# এই steps follow করুন:

# 1. প্রজেক্ট রুটে .env.local নামে নতুন ফাইল তৈরি করুন
# 2. নিচের content copy করুন (আপনার actual values দিয়ে):

NEXT_PUBLIC_SUPABASE_URL=https://your-project-ref.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key_here
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key_here

# Email Configuration (Gmail SMTP)
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
EMAIL_FROM=your-email@gmail.com

# 3. Supabase Dashboard থেকে keys নিন:
#    - Supabase.com এ যান
#    - আপনার project select করুন
#    - Settings > API এ ক্লিক করুন
#    - Project URL copy করুন
#    - anon public key copy করুন  
#    - service_role key copy করুন

# 4. Server restart করুন:
#    npm run dev

# এই steps complete করার পর সব কিছু কাজ করবে!
